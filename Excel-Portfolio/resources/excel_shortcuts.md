# Excel Keyboard Shortcuts Cheat Sheet

## Essential Shortcuts for Excel Productivity

---

## Navigation Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl + Arrow Key` | Jump to edge of data region |
| `Ctrl + Home` | Go to cell A1 |
| `Ctrl + End` | Go to last used cell |
| `Ctrl + G` | Open Go To dialog |
| `F5` | Go To (same as Ctrl+G) |
| `Ctrl + Page Up` | Previous worksheet |
| `Ctrl + Page Down` | Next worksheet |
| `Alt + Page Up` | Move left one screen |
| `Alt + Page Down` | Move right one screen |

## Power User Tips

### Quick Analysis Tool
- Select data → Click Quick Analysis icon (bottom-right) or press `Ctrl + Q`

### Flash Fill
- Type pattern in adjacent column → `Ctrl + E`

### Camera Tool
- Add to Quick Access Toolbar for live picture of ranges

### Watch Window
- Formulas → Watch Window to monitor cells

---

## Customization Tips

### Add to Quick Access Toolbar (QAT)
1. Right-click any command
2. Select "Add to Quick Access Toolbar"
3. Access with `Alt + 1`, `Alt + 2`, etc.

### Customize Ribbon
- File → Options → Customize Ribbon
- Create custom tabs for your most-used commands
